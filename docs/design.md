# 设计

* 基于OpenGL ES 2.0渲染，良好的跨平台支持。
* 渲染层丰富的扩展性，方便实现各种特效，以及进行3D渲染方面的扩展，可以用来开发游戏。
* 多线程架构，确保动画的流畅，以及和原生一样的性能以及耗电标准。
* 内置完善的移动端原生控件，基于Flexbox的自动布局系统，多点触控和手势识别接口。
* 声明式的样式表，React风格的无状态编程风格。
* 使用Lua作为脚本语言。
